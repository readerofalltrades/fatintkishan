# fatintkishan.com

> A Reader of all trades. Curious by Default. Segfaults Occasionally.

**Live:** https://fatintkishan.pages.dev

Personal portfolio of Fatin Tazwar Kishan — CSE student at BRAC University. Built to showcase projects, and document the learning journey.

---

## Stack

- React + Vite + JavaScript
- HTML, CSS
- Cloudflare Pages
- Blog: MDX via `@mdx-js/rollup`

---

## Structure

```
src/
├── components/   # Navbar, Footer, FadeIn
├── sections/     # Homepage sections
├── pages/        # Route-level pages (404, etc.)
├── data/         # projects.js, skills.js
└── styles/       # variables.css
public/           # Static assets — images, SVGs, PDFs
```

---

## Design

- Gruvbox dark hard as base theme, Gruvbox light for light mode
- Typography: Fraunces (display), Inter (body), Fira Code (mono), Noto Sans Bengali
- Homepage: minimal and information-dense. Inner pages: creative and expressive.
- Islamic geometric jali pattern in hero background — Bengal Sultanate inspired
